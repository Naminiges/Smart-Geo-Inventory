# Smart-Geo-Inventory

Lingkup Pekerjaan:
- Pengembangan aplikasi web berbasis Python (Django/Flask/FastAPI).
- Perancangan dan implementasi database PostgreSQL dengan PostGIS.
- Implementasi fitur manajemen stok barang (Create, Read, Update, Delete).
- Integrasi peta digital menggunakan Leaflet.js atau Google Maps API.
- Pembuatan dashboard analitik untuk monitoring stok.

Tujuan Kegiatan:
- Menyediakan sistem inventaris yang terintegrasi dan mudah digunakan.
- Memvisualisasikan lokasi aset atau barang secara spasial.
- Meningkatkan efisiensi pengelolaan stok dan pengambilan keputusan.
